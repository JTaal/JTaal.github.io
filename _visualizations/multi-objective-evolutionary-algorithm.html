3D Multi-Objective Evolutionary Algorithm (NSGA-II)

What is this?

This is a visualization of a Multi-Objective Evolutionary Algorithm (MOEA) solving a problem with three competing objectives (X, Y, and Z).

The Pareto Front

The optimal set of trade-off solutions is called the Pareto Front. Use the Visualization toggle to see metrics in 3D.

  • Green Spheres: The current best-found Pareto Front.
  • Yellow to Red Spheres: Less optimal solutions.
  • Wireframe Mesh: The known, perfect solution for the current goal.
